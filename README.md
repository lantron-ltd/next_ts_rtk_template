#### 拷贝配置文件到根目录
`cp envs/.env.local ./`

#### 安装依赖
`yarn`

#### 启动
`yarn dev`

#### 打开目录
`http://localhost:3000`

#### 部署
`yarn build`    
`yarn start`    

#### 特性
1. 支持typescript
2. 集成sass
3. 支持dotenv
4. 支持pwa
5. 集成了redux(redux-toolkit)
6. 自带了一个counter示例
7. 封装了axios，添加了拦截器
